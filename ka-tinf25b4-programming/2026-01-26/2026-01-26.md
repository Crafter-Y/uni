## Kontrollstrukturen

**Sequenz**

Normale Abfolge von Anweisungen:

```mermaid
flowchart TB
    A --> B
    B --> C
```

Beispiel:

```java
statement1;
statement2;
```

**Verzweigungen**

If Anweisung:

```mermaid
flowchart TD
    subgraph if
        B{condition}
        C[ ]
    end

    A[ ] --> B
    B -->|true| C
    C --> D[ ]
    B -->|false| D
```

Beispiel:

```java
if (condition) {
    statement;
}
```

If-Else Anweisung:

```mermaid
flowchart TD
    subgraph if
        B{condition}
        C[ ]
        D[ ]
    end

    A[ ] --> B
    B -->|true| C
    B -->|false| D
    C --> E[ ]
    D --> E
```

Beispiel:

```java
if (condition) {
    statement1;
} else {
    statement2;
}
```

Switch Anweisung

```mermaid
flowchart TD
    subgraph switch
        B{switch}
        C[ ]
        D[ ]
        E[ ]
        F[ ]
    end

    A[ ] --> B
    
    B -->|case 1| C
    B -->|case 2| D
    B -->|case 3| E
    B -->|default| F

    C --> G[ ]
    D --> G
    E --> G
    F --> G
```

Beispiel:

```java
switch (expression) {
    case match1:
        statement1;
        break;
    case match2:
        statement2;
        break;
    default:
        statement3;
}
```

**Schleifen**

While Schleife

```mermaid
flowchart TD
    subgraph while
        B{condition}
        C[ ]
    end

    A[ ] --> B
    B -->|true| C
    C --> B
    B -->|false| D[ ]
```

Beispiel:

```java
while (condition) {
    statement1;
}
```

Do-While Schleife

```mermaid
flowchart TD
    subgraph while
        B[ ]
        C{condition}
    end

    A[ ] --> B
    B --> C
    C -->|true| B
    C -->|false| D[ ]
```

Beispiel:

```java
do {
    statement;
} while (condition)
```

For Schleife

```mermaid
flowchart TD
    subgraph for
        B[init]
        C{condition}
        D[ ]
        E[increment]
    end

    A[ ] --> B
    B --> C
    C -->|true| D
    D --> E
    E --> C
    C -->|false| F[ ]
```

Beispiel:

```java 
for (int i = 0; i < 10; i++) {
    statement;
}
```
