## Weiter Vererbung

Klassen können von anderen Klassen Attribute und Methoden vererben. z.B.

```java
class Person {
    private int userId:
    private String name;
    private String phoneNo;

    public Person (int userId, String name, String phoneNo) {
        this.userId = userId;
        this.name = name;
        this.phoneNo = phoneNo;
    }
}
```

Eine Benutzergruppe Student oder Staff hätte zusätzliche Felder, wie `matrNo` oder `grade` welche dann einfach durch eine Unterklasse angefügt werden kann:

```java
class Student extends Person {
    private int matrNo;

    public Student(int userId, String name, String phoneNo, int matrNo) {
        super(userId, name, phoneNo);
        this.matrNo = matrNo;
    }
}
```

```java
class Staff extends Person {
    private int grade;

    public Student(int userId, String name, String phoneNo, int grade) {
        super(userId, name, phoneNo);
        this.grade = grade;
    }
}
```

Wenn ein Attribut oder Methode ein Attribut oder Methode aus der Oberklasse überschreibt, nennt man das Polymorphie.

## Exceptions

Eine Exception ist in Java ein Objekt der Unterklasse Throwable, welche geworfen `throw` und gefangen `catch` werden kann.

- `try` stellt einen Codeblock zur Verfügung, mit dem Exceptions und abnormale Abbrüche behandelt werden
- `catch` dem `try`-Block folgen null oder mehrere catch-Klauseln, die bestimmte Exception-Typen abfangen und behandeln.
- `finally` den catch-Klauseln folgt optional ein finally-Block, der hinterher aufräumt. Die Anweisung eines finally-Blocks werden garantiert ausgeführt, gleichgültig, mit welchem Status der try-Block beendet wird.

```java
try {
    // Programmcode der Exceptions erzeugt
}
catch (Exception ex) {
    // Programm der für eine Exception ausgeführt wird
} finally {
    // Wird in jedem Fall ausgeführt.
}
```

Man kann Methoden deklarieren und in der Signatur mit angeben, dass die Methode potentiell eine Exception wirft:

```java
public void openFile(File file) throws IOException {
    // hier kann irgendwie diese Exception geworfen oder nicht gefangen werden
}
```
