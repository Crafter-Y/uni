## Klassen und objekte

**Klassenvariablen und -methoden**

Werden definiert mit dem `static` Keyword. Sie sind nicht an die existenz eines Objektes einer Klasse gebnden. Sie existieren genau einmal. 

Der Zugriff kann über den Klassennamen aufgelöst werden.

```java
public static int numCircles = 0;
```

**Statischer Initialisierer**

Wird ausgeführt, wenn eine Klasse geladen wird:

```java
public class Xyz {
    static {
        ...-
    }
}
```
