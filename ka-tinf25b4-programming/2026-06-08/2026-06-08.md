## Swing Komponenten anpassen

Um das Erscheinungsbild eines Swing Components zu bearbeiten, kann man folgende Methoden implementieren:

- `paintComponent(Graphics)`
- `paintBorder(Graphics)`
- `paintChildren(Graphics)`

z.B. 

```java
public class CustomComponent extends JComponent {
    @Override
    protected void paintComponent(Graphics g) {
        g.drawString("Hello World", 25, 50);
    }

    // ...
}
```

Es ist auch möglich von einem JFrame direkt die `paint(Graphics)` Methode zu überschreiben.

Beim manuellen Zeichnen verwendet man immer ein Koordinatensystem, ausgehend vom oberen linken Rand.
