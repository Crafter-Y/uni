## Strings und Wrapperklassen

Strings sind kein primitiver Datentyp, sondern eine Klasse. Sie müssen aber nicht mit `new` erzegt werden.

Einen Zeilenumbruch kann man mit `\n` einfügen.

String kann man mit `+` verketten:

```java
String s1 = "Develop";
String s2 = "ment";

String s = s1 + s2; // Development
```

Um Strings zu vergleichen verwendet man `.equals(String)` oder `.equalsIgnoreCase(String)`.

Mit `.compareTo(String)`  können Strings lexikografisch verglichen werden. `.contains(String)` überprüft ob eine Zeichenkette in einer anderen enthalten ist.

Mit `.chatAt(int)` kriegt man ein Zeichen an einem Index in einem String.

Mit `.endsWith(String)` prüft man, ob eine String mit einem bestimmten String endet.

Mit `indexOf(String)` bekommt man den index eines Strings in einem String.

`.length()`: Länge der Zeichenkette

`.substring(int, int)`: Liefert einen Teil-String von - bis

`.replace(char, char)`: Ersetzt alle vorkommenden Zeichen durch ein neues

`.valueOf(other)`: Liefert die String-Darstellung des Datentyps bzw. Objekts

`.matches(String regex)`: ermittelt, ob ein String dem regulären Ausdruck regex entspricht

`.replaceAll(String regex, String)`: Ersetzt alle Vorkommen von regex durch einen String

**printf Formatierung**

Mit `String.format()` kann man die bewährte `printf` Funktion verwenden, die man aus unix oder C kennt. [printf man](https://www.man7.org/linux/man-pages/man3/printf.3.html)

**Weitere String Wrapper**

Mit einem `StringBuffer` kann man Strings effizienter machen. `StringBuilder` können Operationen mit Strings vereinfachen.

**Wrapperklassen**

`Void`, `Byte`, `Short`, `Integer`, `Long`, `Float`, `Double`, `Character` existieren, für bestimmte Fälle, an denen man ein Object braucht.

**Autoboxing**

Die Konvertierung in ein Objekt der entsprechenden Wrapper-Klasse wird Boxing genannt.

Die Rückumwandlung in einen primitiven Datentyp heißt Unboxing.

|          | Ausführliche Schreibweise         | Autoboxing       |
| -------- | --------------------------------- | ---------------- |
|          | `int i = 10;`                     | `int i = 10;`    |
| Boxing   | `Integer j = Integer.valueOf(i);` | `Integer j = i;` |
| Unboxing | `int k = j.intValue();`           | `int k = j;`     |
