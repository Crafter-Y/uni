## Grafische Anwendungen mit Swing

Manche Programme benötigen eine grafische Oberfläche. Java Swing ist eine interne Bibliothek, welche so etwas ermöglicht. Swing ist schon etwas älter - reicht aber für vieles bereits aus.

Top-Level Container:

- JFrame (Fenster mit Rahmen)
- Jwindow (Fenster ohne Rahmen)
- JDialog (Dialogfenster)
- JApplet (??)

Innerhalb dieser Top-Level Container gibt es Wurzelkomponenten.

Wichtige Methoden auf Top-Level-Containern:

- setTitle(String)
- add(Component)
- setSize(int, int)
- pack(void) (Passt die Fenstergröße an das aktuelle Layout an)
- setDefaultCloseOperation(int) (Was passieren soll, wenn das Fenster geschlossen wird)
  - WindowConstants.DO_NOTHING_ON_CLOSE
  - WindowConstants.HIDE_ON_CLOSE
  - WindowConstants.DISPOSE_ON_CLOSE
  - WindowConstants.EXIT_ON_CLOSE
- setVisible(boolean)
- dispose(void) (schließt das Fenster und gibt alle Ressourcen frei)

Weitere Container:

- JPanel (Nur für layout wie z.B. &lt;div&gt;&lt;/div&gt;)
- JScrollPane
- JTabbedPane
- JInternalFrame
- JOptionPane (Container füt Dialogfenster)
- JSplitPane
- JToolBar

Wichtige Grudkomponenten:

- JComponent
- JLabel
- JCheckBox, JRadioButton, JToggleButton
- ButtonGroup
- JComboBox, JList
- JTextField, JPasswordField, JTextArea
- JFormattedTextDocument
- JFileChooser
- JMenuBar, JMenu, JMenuItem, JChekboxMenuItem, JPopupMenu, JSeperator, JScollPane
- ImageIcon
- JProgressBar
- JSlider
- JColorChooser
- JToolTip
- JTree
- JTable
- JEditorPane, JTextPane

Das Look and Feel (Design der Oberfläche) kann man einstellen: (eines der folgenden)

```java
UIManager.setLookAndFeel(UIManager.getCrossPlatformLookAndFeelClassName());
UIManager.setLookAndFeel("javax.swing.plaf.metal.MetalLookAndFeel");
UIManager.setLookAndFeel(UIManager.getSystemLookAndFeelClassName());
```

Layout Manager:

- FlowLayout (Standard)
- BoxLayout
- GridLayout
- BorderLayout
- GridBagLayout
- CardLayout
- SpringLayout
- GroupLayout
- NullLayout

Dialoge sind spezielle Fenster, die nur temporär erscheinen, z.B. für Dateiauswahlen oder einen Farbwähler.

Man kann diese Dialoge selber definieren, es gibt aber auch bereits vorgefertigte Dialoge, welche man aufrufen kann, mit:

- showMessageDialog()
- showInputDialog()
- showConfirmDialog()
- showOptionDialog()
