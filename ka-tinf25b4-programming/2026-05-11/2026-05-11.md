## Events

Graphische Anwendungen verwenden Events. Damit kann gewährleistet werden, dass man nicht die ganze Zeit auf Input wartet und nebenbei das Programm weiterläuft.

In Java verwenden wir das Event-Listener (Observable) Prinzip 

Einem Swing Component kann man einen Listener hinzufügen:

```java
JButton button = new JButton("Click me!");
button.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        button.setText(text.getText());
    }
});
```

Es gibt immer die Abwägung, ob man hier eine anonyme oder eine seperate Klasse verwendet. In den meisten Fällen möchte man die Klasse aber nicht wiederverwenden, was für eine anonyme Klasse spricht.

Über `ActionEvent#getSource()` kann man auf das Quellelement zugreifen und mit dessen Attributen (z.B. `JButton`) arbeiten.

## Popup Menus

Anstelle einer `JMenu` kann man auch ein `JPopupMenu` erstellen und Menueinträge (`JMenuItem`) hinzufügen.

Das Popup kann man dann mit `JPopupMenu#show(String, int, int)` zeigen.
