## Arrays

Dienen zum Speichern mehrerer gleichartiger Daten.

Beispiel:

```java
float[] v = new float[10];
```

Arrays sind Objekte, die dynamisch erzeugt werden. Sie selber sind aber nicht dynamisch und müssen mit einer festen Größe erzeugt werden.

Die Länge eines Array bekommt man durch das `.length` Attribut:

```java
float[] v = {1.0f, 1.1f, 1.5f};
System.out.println(v.length); // 3
```

Referenzkopie/Wertkopie:

```java
float[] v = {1f,2f,3f};

float[] q = v; // Kopie als Referenz

float[] r = float[5];

System.arraycopy(v, 0, r, 0, v.length) // Wertkopie
```

**Mehrdiemensionale Arrays**

Beispiel:

```java
byte[][] twoDimArray = new byte[256][16];
```
