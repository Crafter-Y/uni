## GSON

In vielen Web-Anwendungen wird JSON (JavaScript Object Notation) verwendet. Java Anwendungen können mithilfe von Bilbliotheken solche JSON Objekte verarbeiten, also:

- Java-Objekte nach JSON konvertieren ("Serialisierung")
- JSON in Java-Objekte umwandeln ("Deserialisierung")

Die meisten notwendigen Klassen befinden sich im Package `com.google.gson`. 

Die Bibliothek kann man folgendermaßen in gradle einbinden:

```groovy
implementation("com.google.code.gson:gson:2.14.0")
```

Grundlegende Methoden:

- `String toJson(Object)`
- `T fromJson(Object, T.class)`

Beispiele:

```java
System.out.println(gson.toJson("abcd")); // "abcd"

int[] values = { 1 };
System.out.println(gson.toJson(values)); // [1]

Integer someInteger = gson.fromJson("1", Integer.class);
```

Aber auch kompexere Klassen mit verschachtelten Feldern können Serialisiert werden.

Die Standart-Serialisierung liefert einen kompakten String mit einer langen JSON Zeile. Mit Pretty-Printing kann man eine schönere Ausgabe erzeugen:

```java
Gson gsonP = new GsonBuilder().setPrettyPrinting().create();
System.out.println(gsonP.toJson(results));
```

## JavaFX

Bei JavaFX ist das Basis Package `javarfx.*`. Bei einer Anwendung geht man folgendermaßen vor:


- Instanz von `Application` erzeugen
- Methode `start(Stage)` aufrufen
- Am Ende `Platform.exit()` aufrufen

Auf der Stage können eine oder mehrere Scenes platziert werden, die wiederum Nodes enthalten können.

Hello World:

```java
public class HelloWorldExampleFX extends Application {
    @Override
    public void start(Stage primaryStage) {
        Button btn = new Button();
        btn.setText("Say Hello World");

        btn.setOnAction(new EventHandler<ActionEvent>() {
            @Override
            public void handle(ActionEvent event) {
                System.out.println("Hello World!");
            }
        });

        StackPane root = new StackPane();
        root.getChildren().add(btn);
        Scene scene = new Scene(root, 300, 250);
        primaryStage.setTitle("Hello World!");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        Application.launch(args);
    }
}
```

Für JavaFX gibt es grafische Editoren, um Szenen zu erstellen, wie z.B. "Scene Builder"

**Events**

Für Events gibt es die generis che Klasse `EventHandler`, welche per Typ-Parameter ein Event annimt:

```java
EventHandler<ActionEvent> handler = new EventHandler<ActionEvent>() {
    @Override
    public void handle(ActionEvent event) {
        System.out.println("Hello World!");
    }
};
btn.setOnAction(handler);
```

In kurz auch:

```java
btn.setOnAction(_ -> System.out.println("Hello World!"));
```

**Layouts**

In JavaFX gibt es die gleichen Layouts und noch mehr als es in Swing gibt. Für ein BorderLayout verwendet man Beispielsweise ein `BorderPane`:

```java
Text centerText = new Text("Center");
Text topText = new Text("Top");
Text rightText = new Text("Right");
Text bottomText = new Text("Bottom");
Text leftText = new Text("Left");

BorderPane.setAlignment(topText, Pos.TOP_CENTER);
BorderPane.setAlignment(bottomText, Pos.BOTTOM_CENTER);
BorderPane.setAlignment(leftText, Pos.CENTER_LEFT);
BorderPane.setAlignment(rightText, Pos.CENTER_RIGHT);

BorderPane root = new BorderPane(centerText, topText, rightText, bottomText, leftText);

Scene scene = new Scene(root, 300, 250);
```

Styling funktioniert mit einem an CSS angelehntem System:

```java
root.setStyle("-fx-padding: 10; -fx-border-style: solid inside;" +
              "-fx-border-width: 2; -fx-border-insets: 5;" +
              "-fx-border-radius: 5; -fx-border-color: red;" );
```
