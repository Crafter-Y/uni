# Java

Java is... 

> ...a simple, object-oriented, distributed, interpreted, robust, secure, architectural neutral, portable, high-performance, multithreaded, and dynamic language.
>
> Sun Microsystems

Zudem ist Java Plattformunabhängig (Cross-Platform).

Es gibt mehrere Sorten von Java (JDK), u.a.:

- Java SE (Standart Edition)
- Java EE (Enterprise Edition)
- Java ME (Micro Edition)

Java ist angelehnt an andere Programmiersprachen:

- Kein Präprozessor (Aber Annotations)
- Keine Pointer
- Keine überladenen Operatoren, mehrfachvererbung
- Exceptions
- Garbage Collection
- Packages
- Cuncurrency ($\neq$ Paralellism)

## Aufbau eines Programms

Ein Javaprogamm besteht aus Klassen. Die Hauptklasse muss eine `main` Methode enthalten. Das ist der Einstiegspunkt für jedes* Java Programm.

Eine Java Klasse wird in einer `.java` Datei gespeichert. Kompiliert wird es in das `.class` Bytecode Format. Dieses Kompilierte Java Programm kann dann von Java Interpretern/JVMs ausgeführt werden.

## Packages

Packages sidn eine Gruppe von thematisch zusammengehörenen Klassen.

Mit dem `package` keyword setzt man das package einer Klasse:

```java
package mymodule;

public class ClassName {

}
```

Um eine Klasse aus einem Package zu importieren, nutzt man das `import` keyword:

```java
package myothermodule;
import mymodule.ClassName;

class OtherClassName {
    ClassName firstClass;
}
```

Man kann auch alle Klassen eines Packages importieren:

```java
import mymodule.*;
```

Packages benennt man immer klein und ohne Verbindungszeichen
