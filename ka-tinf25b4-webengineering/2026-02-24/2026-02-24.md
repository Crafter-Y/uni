## CSS

CSS ist eine deklatarive Sprache für Stilvorlagen. Man verwendet es, um Struktur und darstellung zu trennen.

**Syntax**

`Selektor {Eigenschaft: Wert; ...}`

Beispiel:

```css
h1 {
    font-family: sans-serif;
    font-size: large;
    color: olive;
}
```

**Einbinden in HTML**

Methode 1: Style Attribut:

```html
<p style="font-family: fantasy;">Abc</p>
```

Methode 2: Style Tag:

```html
<style>
    p {
        font-family: fantasy;
    }
</style>
```

Methode 3: CSS Datei:

```html
<link rel="stylesheet" type="text/css" href="style.css">
```

**Selektoren**

Universalselektor: *

Typselektor: X

Klassenselektor .klasse

ID-Selektor: #id

Attributselektoren:

- `[attribute]`
- `[attr=1]`
- `[attr1][attr]` Attr 1 und attr

Nachfahren: E F

Kinder E > F

Geschwister E ~ F (nur nachfolgende)

Nachbarn E + F

Pseudoklassen: X:pseudo

Box-Modell:

- Margin: Außenabstand
- Padding: Innenabstand
- Border: Rahmengröße
- Height/Width: Elementgröße

`<span>`: Leeres Inline Element

`div`: Leeres Block Element

## Aufgaben

Aufgabe 2:

Gegeben ist folgendes HTML:

```html
<body>
    <h1 id="h1">Meine Überschrift</h1>
    <p id="p1">Absatz 1</p>
    <h2 id="h2">Überschrift 2</h2>
    <p id="p2">Absatz 2</p>
    <ol id="ol1">
        <li id="li1">Aufzählung 1</li>
        <li id="li2">
            Aufzählung<b id="b">2</b>
        </li>
        <li id="li3">Aufzählung 3</li>
    </ol>
    <p id="p3">Absatz 3</p>
    <h2 id="h3">Noch eine Überschrift</h2>
    <p id="p4" class="m k">Absatz 4</p>
</body>
```

Die Elemente mit welchen Ids werden hier selektiert?

- `h2`: h2, h3
- `p li`: keine Treffer
- `ol li`: li1, li2, li3
- `h2 ~ p`: p2, p3, p4
- `h2 > p`: keine Treffer
- `p + p`: keine Treffer
- `h1 ~ h2`: h2, h3
- `ol *`: li1, li2, b, li3
- `ol > *`: li1, li2, li3
- `:first-child`: h1, li1, b
- `:only-child`: b
- `h2:first-of-type`: h2
- `.k`: p4
- `.p4`: keine Treffer
