## CSS

CSS ist eine deklatarive Sprache für Stilvorlagen. Man verwendet es, um Struktur und darstellung zu trennen.

**Syntax**

`Selektor {Eigenschaft: Wert; ...}`

Beispiel:

```css
h1 {
    font-family: sans-serif;
    font-size: large;
    color: olive;
}
```

**Einbinden in HTML**

Methode 1: Style Attribut:

```html
<p style="font-family: fantasy;">Abc</p>
```

Methode 2: Style Tag:

```html
<style>
    p {
        font-family: fantasy;
    }
</style>
```

Methode 3: CSS Datei:

```html
<link rel="stylesheet" type="text/css" href="style.css">
```

**Selektoren**

Universalselektor: *

Typselektor: X

Klassenselektor .klasse

ID-Selektor: #id

Attributselektoren:

- `[attribute]`
- `[attr=1]`
- `[attr1][attr]` Attr 1 und attr

Nachfahren: E F

Kinder E > F

Geschwister E ~ F (nur nachfolgende)

Nachbarn E + F

Pseudoklassen: X:pseudo

Box-Modell:

- Margin: Außenabstand
- Padding: Innenabstand
- Border: Rahmengröße
- Height/Width: Elementgröße

`<span>`: Leeres Inline Element

`div`: Leeres Block Element
