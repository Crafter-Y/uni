## JQuery

JQuery ist eine JavaScript-Bibliothek zur Vereinfachung der browserunabhängigen Manipulation von HTML-Dokumenten.

Lange Zeit war es der go-to Standart für Webseiten mit Logik. Auch heutzutage ist JQuery noch in vielen Webseiten eingebunden. 

**Selektieren und Filtern**

`$` ist das JQuery Objekt. Mit der Funktion `$(<selector>)` kann man fast alle CSS Selektoren und einige JQuery Selektoren verwenden. Zurück erhält man ein JQuery Objekt für das selektierte Element oder eine Liste dieser Elemente. 

Ausgehend von diesem Selektor gibt es dann Filterfunktionen, um die Auswahl einzugrenzen oder um von der Auswahl ausgehend andere Elemente auszuwählen:

| Funktion            | Beschreibung                                                                                     |
| ------------------- | ------------------------------------------------------------------------------------------------ |
| `eq(n)`             | Nur das Element mit Index n                                                                      |
| `slice(n,m)`        | Die Elemente mit Indices zwischen n und m (exkl. m)                                              |
| `first(), last()`   | Erstes, letztes Element                                                                          |
| `has(x)`            | Die Elemente, die einen Nachfahren x haben (x ist Selektor, DOM-Element oder jQuery-Objekt)      |
| `children(s)`       | Kinder                                                                                           |
| `find(s)`           | Nachfahren                                                                                       |
| `parent(s)`         | direkte Eltern                                                                                   |
| `parents(s)`        | Vorfahren                                                                                        |
| `closest(s)`        | Nächster passender Vorfahre                                                                      |
| `parentsUntil(s)`   | Vorfahren bis zum nächsten passenden (ohne diesen)                                               |
| `siblings(s)`       | Alle Geschwister                                                                                 |
| `prev(s)`           | Vorgänger (Einer)                                                                                |
| `prevAll(s)`        | Vorgänger (Alle)                                                                                 |
| `prevUntil(s)`      | Vorgänger bis zum nächsten passenden (ohne diesen)                                               |
| `next`              |                                                                                                  |
| `nextAll`           |                                                                                                  |
| `nextUntil`         |                                                                                                  |
| `filter(f), not(f)` | Nur (nicht) die Elemente die dem Filter f entsprechen.                                           |
| `map(f)`            | Für jedes Element wird die Funktion f aufgerufen und das Element durch den Rückgabewert ersetzt. |

**Manipulieren**

Auf einem JQuery Element stehen dann Methoden bereit um dieses zu verändern:

- `html()`: Lesen des HTML-Inhalts des ersten Elements
- `html(htmlString)`: Schreiben des HTML-Inhalts aller Elemente
- `text()`: Lesen des Text-Inhalts des aller Elemente
- `text(textString)`: Schreiben des HTML-Inhalts aller Elemente
- `attr()`: Lesen oder Setzen eines Attributes
- `val()`: Lesen oder Setzen des Wertes eines Form-Elementes
- `width()`: Lesen oder Setzen der Breite
- `height()`: Lesen oder Setzen der Höhe
- `position()`: Lesen der Position, relativ zum nächsten positionierten Vorfahren.

Mit JQuery kann man auch Elemente verschieben oder neu erstellte Elemente an einer bestimmten Stelle einfügen:

- `appendTo(), append()`: Anhängen an die Liste der Kinder

Beispiel:

```js
$("#myList li:first").appendTo("#myList");
$("#myList").append($("#myList li:first"));
```

- `prependTo(), prepend()`: Voransetzen an der Liste der Kinder
- `insertAfter(), after()`: Einfügen nach
- `insertBefore, before()`: Einfügen vor

**Erzeugen und Kopieren**

Mit der `$()` Funktion kann man auch Elemente erstellen:

```js
$("<p>Dies ist ein neuer Absatz</p>")
$( "<a/>", {html: "This is a link", href: "foo.html"});
```

Man kann auch bestehende Elemente kopieren, um sie an anderer Stelle einzufügen:

```js
const newP = $("p").first().clone();
```

**Elemente löschen**

Es gibt verschiedene Methoden zum Löschen:

- `remove()`: Entfernen der Selektion vom DOM, Löschen von Daten und Handlern
- `detach()`: Entfernen der Selektion vom DOM, Erhalten von Daten und Handlern
- `empty()`: Entfernen aller Kindknoten
- `unwrap()`: Entfernen aller Elternknoten

**Iteration über eine Selektion**

Mit `each()` kann man über alle Elemente einer Selektion iterieren:

```js
$("p").each((index, listItem) => {
    // do something with listItem
});
```

**Events**

Mit `on()` kann man einen EventHandler auf ein Element hinzufügen. Dabei kann man auf die gleichen Events hören, die auch in der `addEventListener` API zur Verfügung stehen. 

```js
$("#myElement").on("click", (event) => { /* ... */ })
```

**JQuery UI**

JQuery UI bietet neben Komponenten (sog. Widgets) bietet die Library auch Interaktionen, Effekte und Utilities an.

So kann man z.B. mit nur einem Selektor ein Datepicker Input Element zu einem Datepicker machen:

```js
$("#datepicker").datepicker();
```

Nach dem gleichen Prinzip gibt es auch:

- Accordion
- Autocomplete
- Button
- Checkboxradio
- Controlgroup
- Dialog
- Menu
- Progressbar
- Selectmenu
- Slider
- Spinner
- Tabs
- Tooltip

Für Tabellen oder Listen gibt es:

```js
$("#sortable").sortable();
```

Und für sonstige Elemente Utilities, wie:

- Draggable
- Droppable
- Resizable
- Selectable
