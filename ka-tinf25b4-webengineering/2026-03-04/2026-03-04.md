# CSS

**Transformationen**

Mit `transform:` kann man Elemente verändern:

- `translate(30px, 50px)`
- `rotate(30deg)`
- `scale(0.75, 2.5)`
- `skew(30deg)`

Diese sachen kann man auch kombinieren:

z.B. `transform: translate(70px, 50px) rotate(45deg) skew(30deg);`

Das ganze gibt es auch in 3d `translate3d`, `matrix3d` ...

**Anchor positioning**

- `anchor-name`: Element auf das sich bezogen wird
- `position-anchor`: Das zu positionierende Element

Man verwendet hier `position: absolute` oder `position: fixed`. Dann kann man über die `anchor` Referenz das Element positionieren:

```html
<body>
    <div id="myBase">
        Dies ist der Anker.
    </div>
    <div id="myElement">
        Dies wird positioniert.
    </div>
</body>
```

```css
#myBase { 
    position: absolute;
    top: 200px;
    left: 200px;
    width: 100px;
    height: 100px;
    border: 2px solid red;
    anchor-name: --myAnchor;
}
#myElement { 
    border: 1px solid green;
    position-anchor: --myAnchor;
    position: fixed;
    left: anchor(right);
    top: anchor(top);
}
```

Man kann `anchor-size(height/width)` verwenden, um die Höhe/Breite des Anchorelements zu bekommen

- `position-area`: reihe / spalte
- `position-try-fallback`: Fallback, falls in der angegebenen area kein Platz ist

Fallback Taktiken kann man auch manuell definieren

**Media Queries**

`@media <medium>`, z.B. print, speech, braille, handheld

**Important**

Wenn man hinter eine CSS Regel `!important` schreibt, dann hat sie Priorität.
