# HTML

Der Kurs wird sich nicht auf bestimmte moderne Framerworks konzentrieren, sondern auf die Grundlagen eingehen.

Wir werden uns befassen mit:
- HTML
- CSS
- JavaScript
- JSP
- (HTTP)

Es gibt verschiedene Versionen von HTML: XHTML (1.1), HTML 4 und HTML 5. Wir arbeiten mit html5. 

In HTML gibt es sog. Tags. Diese gibt es für Überschriften, Absätze, Verweise usw. Dafür steht "Hypertext" und es ist kein neues Prinzip.

Die meisten Tags müssen geöffnet und geschlossen werden:

`<tag>` und `</tag>`

Manche tags müssen nicht geschlossen werden:

`<br>`

Manche Tags kann man aber auch direkt schließen

`<tag />`

*Attribute*

Man kann werte in Tags definieren:

`<tag attr="abc" />`

Tags kann man verschachteln:

`<tag1><tag2></tag2></tag1>`

*Aufbau eines Dokuments*

```html
<!DOCTYPE html> <!-- HTML5 -->
<html>
    <head>
        <meta charset="utf-8">
        <title>Insert title here</title>
    </head>
    <body>
        ...
    </body>
</html>
```

Das muss zwar alles nicht vorhanden sein (vom Browser her), wird aber als "Grundlage" erwartet.

## Tags

- `<p>Text</p>`: Absatz mit neuer Zeile starten und danach einen Absatz
- `<h1>Überschrift</h1>`: h1-h6 - Überschriften werden kleiner. Davor und danach einen Absatz
- `<b>Fett</b>` alternativ `<strong>`
- `<i>Italic / kursiv</i>` alternativ `<em>`
- `<sup>Hochgestellt</sup>`
- `<sub>Tiefgestellt</sub>`

*Wichtige Attribute*:

- id: Eindutiger Name ohne Leerzeichen
- class: Zuordnung zu einer Gruppe, z.B. für Styles oder Scripts
- lang: Sprache für das Element und dessen Inhalt
- title: Zusammenfassung für Tooltip und Barrierefreiheit

*Weitere Attribute*:

- accesskey
- dir: Schribrichtung
- tabindex: Rihenfolge beim springen durch Tabs
- spellcheck: Rechtschreibprüfung aktivieren/deaktivieren
- hidden: Element ausblenden
- contenteditable: Element als editierbar setzen.
- translate: Übersetzen erlauben/verbieten
- draggable/dropzone: Für Drag & Drop
- style: Style direkt im Dokument

## Entitäten

Manche Zeichen sind von HTML reserviert und können deshalb nicht verwendet werden.
Dafür gibt es sog. Eintititäten um diese doch als Text einbinden zu können:

- &amp;quot; "
- &amp;lt; <
- &amp;gt; >
- ...

## Listen

Unsortiert:

```html
<ul>
    <li>Milch</li>
    <li>Eier</li>
    <li>Mehl</li>
</ul>
```

Sortiert:

```html
<ol>
    <li>Todo 1</li>
    <li>Todo 2</li>
    <li>Todo 3</li>
</ol>
```

Attribute:

- reversed: Zählung umdrehen
- start: Startwert, z.B. 2
- type: Art des Zählers ändern, z.B. A,B,C oder I,II, III

Definitionsliste:

```html
<dl>
    <dt>Titel 1</dt>
    <dd>Definition 1</dd>
    <dt>Titel 2</dt>
    <dd>Definition 2</dd>
</dl>
```
