## Java Script

Javascript hat begonnen als klassenlose Scriptsprache und ist nun die meistverwendete Programmiersprache welche durch ECMA standartisiert wurde.

**Datentypen und Variablen**

Es gibt grundsätzlich folgende Datentypen in JavaScript:

- object (Umfassed ebenfalls classes)
- array
- number
- string
- boolean
- null
- undefined
- function (Prinzipiell ebenfalls object0)

Zum deklarieren einer Variable gibt es folgende Möglichkeiten

- `a = "Hello World"` (Einfache Zuweisung)
- `const a = 123` (Konstante)
- `var a = null` (Alte Schreibweise für mutable. Schiebt die Variable in den überliegenden Kontext (Hoisting))
- `let a = [1,2,3]` (Klassische Variablendeklaration)

In JavaScript können alle klassischen Rechenoperationen (Arithmetisch und Binär) durchgeführt werden. Dabei versucht JavaScript in jedem Fall eine Typenkonvertierung (Type Coersion).

Kontrollstrukturen wie `for`, `while`, `do-for`, `if`, `return`, `break`, `continue` und `switch-case-default` sind in gewohnter Form verfügbar. 

Funktionen definiert man klassicherweise:

```js
function functionname(parameter1, parameter2){
    // ...
    return something;
}
```

Seit ES7 ist auch der neuere (meist kompaktere) Arrow-Syntax verfügbar:

```js
const functionname = (parameter1, parameter2) => {
    // ...
    return something;
}
```

Objekte einer Klasse können wie gewohnt per `new` Operator instanziiert werden. Da es in der Ursprünglichen Spezifikation von JavaScript keine Klassen gab, hat man funktionen als Klassen verwendet und konnte per `this` Referenz einzelne Felder setzen. 

**JavaScript im Browser**

Im Browser stehen mehrere globale Objekte zur verfügung, z.B.:

- window
- document
- date
- forms
- element

Es gibt auch eine ganze Menge an Standart-APIs, wie z.B. `alert` oder `eval`.

Suchen eines (HTML-) Elementes:

- `document.getElementById()`
- `document.getElementsByName()`
- `document.getElementsByTagName()`
- `document.querySelector("<css-Selektor>");`
- `ocument.querySelectorAll("<css-Selektor>");`

Erzeugen eines neuen (HTML-) Elements

- `document.createElement(tagName)`
- `document.createTextNode(text)`

Man kann JavaScript entweder direkt im HTML schreiben:

```html
<script type="text/javascript">
    // Code
</script>
```

oder als externe `.js` Datei:

```html
<script type="text/javascript" src="code.js"></script>
```

**Events**

Man kann auf einem Objekt auf Events horchen:

```js
element.addEventListener("click", event => someFunction());
```

oder auch direkt auf dem Element. Dabei kommt noch der `on` Prefix dazu:

```html
<p onclick="handler(event, p1, p2);" >...</p>
```

Es gibt viele Elemente, unter anderem:

- onfocus, onchange, onblur, onabort,
- onkeydown, onkeyup, onkeypress,
- onclick, ondblclick
- onmouseover, onmousedown, onmouseup, onmousemove
- ...
