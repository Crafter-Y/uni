Formular POST:

1. Browser schickt URL an Server
2. Server bestimtm die Seite
3. Server schickt Seite an Browser
4. Browser stellt Formular dar
5. Nutzer klickt auf Senden. Browser schickt URL ohne Anfrageparameter, aber mit Dokument mit Parametern an Server
6. Server bestimmt zweite Seite
7. Server schickt zweite Seite zurück
8. Brwoser stellt zweite Seite dar.

Für ein Formular braucht mein ein `form` Element

Attribute:

- method: `GET` oder `POST` (auch andere noch)
- action: Ziel
- name
- target: `_blank` oder `_self`
- autocomplete
- novalidate
- enctype
- accept-charset
- ...

**Eingabefelder**

`input` Element. Inline Element ohne closing tag.

Attribute:

- autocomplete
- autofocus
- required
- pattern: regex
- maxlength
- size
- placeholder
- value
- dirname: Richtungsparameter
- list: id einer datalist
- type: `hidden`, `password`, `date`, `number`, `submit` (und viele mehr)
