## JSP - Tags

Anstelle von PHP-like Syntax wie:

```jsp
<h1>Zahlen</h1>
<% for (int i = 0; i < 100; i++) { %>
    <p>Nummer <%= i %></p>
<% } %>
```

Gibt es auch eingebaute Tags:

```jsp
<c:forEach begin="0" end="99" var="i">
    <p>Nummer <c:out value="${i}" /></p>
</c:forEach>
```

Mit Ausnahme von den JSP Standardtags müssen diese Tags müssen explizit eingebunden werden:

```jsp
<%@ taglib prefix="präfix" uri="https://..." %>
```

- `jsp:element`: HTML Elemente dynamisch definieren
- `jsp:include`: Andere JSP Datei in Laufzeit einbetten
- `jsp:forward`: Anfrage an andere JSP weiterleiten
