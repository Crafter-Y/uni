## Tags

Tags bieten eine Erweiterung zu den bisherigen Möglichkeiten. Sie können Fragmente aus HTML und Skriplets ersetzen und bieten die Möglichkeit eine Art Bibliothek für parametrisierte Komponenten.

Sie bieten abhilfe zu den Expressions, die den Code schnell unübersichtlich werden lassen:

```jsp
<c:forEach begin="0" end="99" var="i" >
    <p>Nummer <c:out value="${i}"/></p>
</c:forEach>
```

Die Standard Tag Library (JSTL) muss man manuell einbinden:

```jsp
<%@ taglib prefix="c" uri="jakarta.tags.core" %>
```

Folgende Tags werden bereitgestellt:

- `<c:set var="x">5</c:set>` - Ein Attribut `scope` kann gesetzt werden, z.B `page`, `session`, `request` oder `application`
- `<c:out value="${'<tag>'}"/>`
- `<c:if test="${name == 'Peter'}">...</c:if>`
- c:choose außen drum, c:when mit Attribut test und c:otherwise als default case
- `<c:forEach begin="1" end="10" var="i" >...</c:forEach>`
- `<c:redirect url="mypage.jsp"><c:param name="x" value="${myX}"/></c:set>`

Es können auch Custom Tags definiert werden.

**Expression Language**

Bietet Zugriff auf Expressions im klassischen String-Interpolation Style: `${expression}`

Beispiele:

- `${x+5*5.7}`
- `${myBean.x}`
- `${m[‘myKey‘]}`
  
Im Gegensatz zu den klassischen Expressions sind folgende Objekte hier implizit verfügbar:

- `cookie`
- `header, headerValues`
- `param, paramValues`
- `initParam`
- `pageContext`
- `pageScope`
- `requestScope`
- `sessionScope`
- `applicationScope`

## Beans

Beans sind Java Klassen, in welche Logik ausgelagert werden kann. Auf sie kann von JSP aus zugegriffen werden. Sie bieten Isolation von Logik und Layout. 

Beans haben keinen Konstruktur. Auf Werte kann man über Getter/Setter Methoden zugegriffen werden, z.B:

```java
public class TemperatureBean {
    private double celcius;

    public TemperatureBean() {
        this.celcius = 0.0;
    }
    public double getCelcius() {
        return celcius;
    }
    public double getFahrenheit() {
        return celcius*1.8 + 32;
    }
    public void setCelcius(double celcius) {
        this.celcius = celcius;
    }
    public void setFahrenheit(double fahrenheit) {
        this.celcius = (fahrenheit -32) *5.0/9.0;
    }
}
```

In JSP mit:

```jsp
<jsp:useBean id="temp" class="beans.TemperatureBean"></jsp:useBean>
...
<jsp:setProperty name="temp" property="fahrenheit" param="f"/>
...
<jsp:getProperty name="temp" property="celcius"/>
```

Auch bei `jsp:useBean` kann ein `scope` gesetzt werden, um den Gültigkeitsberech festzulegen.
