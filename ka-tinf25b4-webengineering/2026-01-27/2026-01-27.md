## Interaktive Elemente

**Details**

Das `details` Element bringt die Möglichkeit eine Collapsable Section zu erstellen:

```html
<details>
    <summary>Expandable Header</summary>

    <p>Hidden contents</p>
</details>
```

Attribute:

- `open`: Standartmäßig geöffnet anzeigen
- `name`: Gruppierung, immer nur 1 Element geöffnet mit dem gleichen Namen

**Dialog**

```html
<button onclick="dialog1.open()">Open Dialog</button>
<dialog id="dialog1">
    <p>Dialog contents</p>
</dialog>
```

**Popover**

```html
<input type="button" value="Popover öffnen" popovertarget="popover1" />
<div id="popover1" popover>
    <p>Popover contents contents</p>
</div>
```

Attribute:

- `popover`: Definiert ein Element als popover
- `popovertarget`
- `popoveraction`
