## CSS

Moderne layouts:

**Anzeigen oder nicht anzeigen**

- `visibility`: visible, hidden, collapse
- `display`: hidden, block, flex, ...

**Umordnen mit float**

- `float`: left, right, none

**Koordinatenbasierte Positionierung**

- `position`: static (Standard), absolute (bzgl static Parent), relative (bzgl parent), fixed (bzgl Window), sticky (fixed after scroll)
- `z-index`: Element in den Vordergrund/Hintergrund stellen
