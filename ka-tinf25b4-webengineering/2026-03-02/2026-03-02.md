## CSS

Moderne layouts:

**Anzeigen oder nicht anzeigen**

- `visibility`: visible, hidden, collapse
- `display`: hidden, block, flex, ...

**Umordnen mit float**

- `float`: left, right, none

**Koordinatenbasierte Positionierung**

- `position`: static (Standard), absolute (bzgl static Parent), relative (bzgl parent), fixed (bzgl Window), sticky (fixed after scroll)
- `z-index`: Element in den Vordergrund/Hintergrund stellen

**Flexbox**

Ein Flex Container bekommt `display: flex`

- `flex-wrap`: `nowrap`, `wrap`, `wrap-reverse`, `inherit`, `initial`
- `flex-direction`: `row`, `column`, `row-reverse`, `columns-reverse`

Ausrichten:

- `justify-center`: `flex-start`, `flex-center`, `space-between`, `space-around`, ...
- `align-items`: `flex-start`, `flex-end`, `flex-center`, `baseline`, ...
- `align-self` für einZelnes Item

Abstände:

- `row-gap`: Abstand 
- `column-gap`: Abstand
- `gap`: Kurzform

Weitere:

- `order`: Reihenfolge eines Elements manuell ändern
- `flex-basis`: Basis-Breite eines Elements
- `flex-grow`: Wachstumsfaktor
- `flex-shrink`: Schrumpffaktor
- `flex`: Kutzschreibweise für grow, shrink und base

**Grid Layout**

Ein Grid Container bekommt `display: grid`.

- `grid-template-columns`: `auto auto` für 2 Spalten mit automatischer Breite
- `grid-template-rows`. Alternativ auch für Zeilen

Ähnlich, wie bei Tabellen, können einzelne Elemente auch mehrere Zeilen oder Spalten einnehmen:

- `grid-row-start`: Startzelle
- `grid-column-start`: Startzelle
- `grid-row-end`, `grid-column-end`, `grid-row`, `grid-column`, `grid-area`
