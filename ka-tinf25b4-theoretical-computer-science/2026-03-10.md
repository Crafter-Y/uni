## Prolog

**Arithmetik**

Die arithmetische Auswertung von Ausdrücken erfolgt in Prolog mit dem `is` Operator. Außerdem gibt es die folgenden arithmetischen Vergleichsoperatoren:

- `X =:= Y`: X ist arithmetisch gleich Y
- `X < Y`
- `Y > Y`
- `X =\= Y`: X arithmetisch nicht gleich Y
- `X =< Y`
- `X >= Y`

Zudem gibt es diese Operanden und Standardfunktionen:

| Beschreibung            | Ausdruck      |
| ----------------------- | ------------- |
| Addition                | `X + Y`       |
| Subtraktion             | `X - Y`       |
| Negation                | `-X`          |
| Multiplikation          | `X * Y`       |
| Division                | `X / Y`       |
| Ganzzahldivision        | `X // Y`      |
| Rest bei Division       | `X mod Y`     |
| Betragswert             | `abs(X)`      |
| Vorzeichen              | `sgn(X)`      |
| Mathematische Rundung   | `round(X)`    |
| Abschneid Rundung       | `truncate(X)` |
| Abrundung               | `floor(X)`    |
| Aufrundung              | `ceil(X)`     |
| Potenzierung            | `X ** Y`      |
| Quadratwurzel           | `sqrt(X)`     |
| Exponation              | `exp(X)`      |
| Natürlicher Logarithmus | `log(X)`      |
| Sinus                   | `sin(X)`      |
| Cosinus                 | `cos(X)`      |
| Arcustangens            | `atan(X)`     |
| Bit Komplement          | `\ X`         |
| Bitshift links          | `X << Y`      |
| Bitshift rechts         | `X >> Y`      |
| Bitweises Und           | `X /\ Y`      |
| Bitweises Oder          | `X \/ Y`      |

Beispiel:

```prolog
hypot(X,Y,Z) :- Z is sqrt((X ** 2) + (Y ** 2)).
```

**Listen und Rekursion**

Listen in Prolog sind eine endliche Liste von Elementen. Lsiten können durch eckige Klammern, durch den Punkt-Operator oder der Bracket-Notation mit senkrechtem Strich elementeweise oder mit mehreren Elementen auf einmal aus einer leeren Liste [] erzeugt werden.

Elemente von Listern können auch Listen sein, so dass auch mehrdimensionale Listen umgesetzt werden können. In der typrischen Unifizierung

$[H|T] = [a_1, a_2, ..., a_n]$

wird `H = a_1` als Head und `T = [a_2, ..., a_n]` als Tail der Liste bezeichnet.

**Cut**

Zur Berechnung der Fakultät kann diese rekursive Prozedur verwendet werden:

```prolog
factorial(N,F)   :- factorial (N,1,F ).
factorial(0,P,F) :- F is P.
factorial(N,P,F) :- N>0, Q is P*N , M is N-1, factorial (M,Q,F).
```

Die Überprüfung auf `N>0` ist erforderlich, da sonst die letzte Regel unendlich oft wiederholt wird. Dies kann aber mit einem sogenannten **Cut** verhindert werden: Dieser verhindert das versuchte Backtracking zu weiteren Fällen. Dann kann der Code etwas vereinfacht werden:

```prolog
factorial(N,F)   :- factorial (N,1,F ).
factorial(0,P,F) :- F is P,!.
factorial(N,P,F) :- Q is P*N , M is N-1, factorial (M,Q,F).
```

**Graphen**

Ein Graph besteht aus einer Menge von Knoten und Kanten. Es gibt eine Kante zwischen zwei Knoten $k_1, k_2 \in V$, wenn $(K_1, K_2) \in E$. Ist E symmetrisch, so folgt aus $(a,b) \in E$ auch $(b,a) \in E$, so ist der Graph **ungerichtet**, sonst wird der Graph als **gerichtet** bezeichnet. Gibt es eine Abbildung $w: E \rightarrow \mathbb{R}$, so ist der Graph **gewichtet**.

**If-Then-Operator**

Der If-Then-Operator `X−>Y` nimmt der Wert des Ausdrucks Y an, wenn X zutrifft und ist sonst false. Der If-Then-Else-Operator `X−>Y;Z` nimmt den Wert von Y an, wenn X zutrifft, und sonst den Wert von Z.

## Haskell

Typen in Haskell:

- Bool
- Int
- Integer
- Float
- Double
- Char
- String

Literale können mit Typ angegeben werden, z.B. `12::Double`

Man kann von Elementen den Typ erfragen:

`:t True` => `true :: Bool` (True ist vom Typ Bool)

Variablen/Platzhalter kann man einfach zuweisen:

`zahl = 5`

**Ausdrücke** in Haskell sind: Literale (wie `12.0`), Platzhalter (wie `pi`), Operanden (wie `(2+4)`) oder Funktionen (wie `max 3 7`).

Parameter für Funktionsaufrufe werden in Haskell hinter die Funktionen statt in Klammern geschrieben.

Operanden in Haskell:

- `==` und `/=`: Gleichheit und Ungleichheit
- `<=`, `>=`, `<`, `>`: Größenvergleich
- `||` und `&&`: Logische Operatoren
- `+`, `-`, `*`, `/`, `**`: Arithmetische Operatoren
- `.`: Verkettung zweier Funktionen

**Funktionen**

In Haskell sind Funktionen auch Werte und haben deshalb auch einen Typen.

Beispiel:

- `:t (==)` => `Eq a = > a -> a -> Bool`
- `:t (==) 5` => `(Eq a , Num a) => a -> Bool`
- `:t (==) 5 3` => `Bool`
- `:t (<=)` => `Ord a => a -> a -> Bool`
- `:t (/)` => `Fractional a => a -> a -> a`
